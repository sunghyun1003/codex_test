# Banner Reference Style Notes

The local `banner_sample/` images were reviewed as creative references, not as model-training data.

## Shared Direction

- Use square 1:1 feed-style composition, close to 1080x1080.
- Use large, bold Korean headline typography as the first visual anchor.
- Keep a clear two-step hierarchy: headline first, support copy second, CTA last.
- Place the CTA as a strong bottom bar, pill button, or clearly separated lower action area.
- Keep the core object simple and readable: car, driver, phone UI, checklist, character, or map-like scene.
- Use varied treatments by concept: real photo, 2D illustration, 3D/digital illustration, cartoon, and editorial collage.
- Avoid overly empty minimal layouts; reference banners are direct, punchy, and ad-like.
- Avoid cash, coins, gold, crypto, price tags, and premium-price wording.

## Style Families To Reuse

- Real vehicle photo with bold headline and CTA bar.
- Character or mascot-led explanatory insurance banner.
- Cartoon problem-solution scene with large emotional headline.
- Clean comparison/checklist layout with strong contrast.
- Isometric or 3D service scene for mobile/app-like concepts.
